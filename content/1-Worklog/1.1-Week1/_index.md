### Week 1 Achievements

* Gained a basic understanding of AWS and its core service categories (Compute, Storage, Networking, Database, etc.).
* Successfully created an AWS account and received AWS Credits from the program.
* Became familiar with the AWS Management Console and the program guidelines.
* Completed hands-on practice with AWS Identity and Access Management (IAM).
* Created and managed IAM Users, Groups, and Roles, and assigned permissions using IAM Policies.
* Understood the principle of least privilege and configured basic security settings for the AWS account.
* Connected with other participants, collaborated with teammates, and formed a project group.
