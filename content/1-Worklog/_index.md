## Introduction

During the **12-week internship**, I participated in an **Amazon Web Services (AWS)** training program while contributing to the development of **AuraAcademic – An AI-powered Online Examination Monitoring System**.

Throughout the internship, I studied and gained hands-on experience with core AWS services, including **Amazon EC2, Amazon VPC, Amazon S3, Amazon RDS, AWS Lambda, Amazon API Gateway, Amazon CloudWatch, Amazon CloudFront, Amazon ECS, AWS WAF, AWS Secrets Manager, Amazon Cognito**, and AWS **CI/CD** services. I applied this knowledge to design, deploy, and optimize the cloud architecture for the AuraAcademic project.

In addition to learning AWS technologies, I actively communicated with the program administrators, received valuable guidance, and continuously improved my technical skills. I also participated in designing the system architecture, developing project features, collaborating with team members, performing system testing, optimizing application performance, and completing the final product.

The following sections summarize my work and achievements throughout the **12-week internship**.
